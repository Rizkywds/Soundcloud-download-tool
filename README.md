#Soundcloud download tool
只是一个萌新的自用小工具罢了。是四年前写的,至今仍在维护（不过其实变化也不多,看得出来sc的安全性很堪忧）

#原理
解析HAR文件,并将被分块的音频重组。

#用法
1.使用Chromium内核的浏览器打开Soundcloud中你想下载的歌曲页面,打开F12调试,按Ctrl + Shift + R 强制刷新。然后点击播放（如果没有自动播放的话）,点开"资源"栏,等待"data0xx.m4s"的文件全部下载好（下载速度很快,几秒钟没有新文件加入就是下载好了）,点击右上角的下载按钮（或者右键->保存为HAR文件之类的）。退出之前右键一个data0xx.m4s,选择复制链接。
2.打开soundcloud2.html。截取复制的链接的前部分".../data",粘贴到弹窗中。然后导入文件，如果出现"下载原神"的按钮说明解码成功，点击下载即可。
3.如果报错，很大概率是下载的HAR文件有问题，多试一两次可以解决。

#灵感
原先作者只能通过保存HAR,用Fiddler解包,再用AE还是啥的软件拼接音频的方式来保存soundcloud线上播放的内容。但是作者歌瘾极重,这种缓慢的速度已然完全不够用,所以有了这个自动化软件。
